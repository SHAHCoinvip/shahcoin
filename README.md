### Quick Start

```bash
# Clone the repository
git clone https://github.com/SHAHCoinvip/shahcoin
cd shahcoin

# Build Qt Wallet
make -f Makefile.unix

# Run the wallet
./src/qt/shahcoin-qt
```

### Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
